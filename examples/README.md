## Using these templates

**As-is**: Copy a script and paste it into the MMBRC platform.

**As a starting point**: Modify labels, add/remove actions, change params to fit your test case.

**With an agent**: Point your agent at these examples as few-shot context, or use the `/generate-fbdl` skill to create new scripts from natural language.
